Intranet-Interface

A user-interactive interface for browsing through Intranet. Intranet Content Inventory

Pre-requisites Local web server, XAMPP, for execution of PHP and MySQL database files.
Execution Clone or download the repository to the htdocs folder of your local web server.
git clone https://github.com/raoniz/Intranet-Interface.git
Start Apache and MySQL service in the XAMPP Control Panel.
Goto 'localhost/phpmyadmin' in your web browser.
Create database intranet_db.
CREATE DATABASE intranet_db
Import database file 'database/intranet_db.sql' into the newly created database (intranet_db).
Goto 'localhost/Intranet-Interface' in your web browser.

Lo! Intranet is live..
